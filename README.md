# Database Diagram Overview

The database consists of three main tables it's users, vehicles and bookings tables

## Users
Stores user and admin information like user_id, name, email, password, phone, role

## Vehicles
Stores vehicle details like vehicle_id, name, type, model, registratons_number, rental_price, status

## Bookings
Stores booking like booking_id, user_id, vehicle_id, start_date, end_date, status, total_cost
