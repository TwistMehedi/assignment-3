# EDR and SQL Queries and video viva Project 


## Project Overview
This project is a SQL based Vehicle Rental Management System designed to demonstrate core database concepts such as edr, queries and video viva.


# Database Diagram Overview

The database consists of three main tables it's users, vehicles and bookings tables

## Users
Stores user and admin information like user_id, name, email, password, phone, role

## Vehicles
Stores vehicle details like vehicle_id, name, type, model, registratons_number, rental_price, status

## Bookings
Stores booking like booking_id, user_id, vehicle_id, start_date, end_date, status, total_cost

# Queries Explanation
All queries are written and stored in the queries.sql file. Each query is designed based on a specific requirement.

## Query Types Covered
 INNER JOIN,
 EXISTS / NOT EXISTS,
 WHERE filtering,
 GROUP BY & HAVING.
